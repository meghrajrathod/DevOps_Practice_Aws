# FinVerse

Enterprise Banking & Finance Platform built for advanced DevOps learning.

## Tech Stack

- ASP.NET Core
- PostgreSQL
- Docker
- AWS ECS
- Terraform
- Jenkins
- GitHub Actions
- CloudWatch
- SNS
- SQS
- SES
- Route53
- CloudFront
- IAM
- VPC
- Prometheus
- Grafana

## Goals

- Learn real-world DevOps
- Build production-grade AWS infrastructure
- Practice CI/CD
- Practice Infrastructure as Code
- Implement monitoring and scaling
- Learn enterprise cloud architecture

## Project Structure

- backend → ASP.NET Core APIs
- frontend → UI application
- database → database scripts
- terraform → Infrastructure as Code
- ansible → configuration management
- docker → containerization
- monitoring → Prometheus/Grafana configs
- scripts → automation scripts
- docs → architecture and documentation
